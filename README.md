# 🎴 3D Perspective Card: Interactive TikTok-Style Profile HUD

3D Perspective Card waa bog loogu talagalay soo bandhigista profile shakhsi ah oo qaabkeedu u jaysan yahay kaadhadhka casriga ah ee TikTok ama ciyaaraha (Gaming Cards). Boggan wuxuu isku dhex dhabbaa **CSS 3D Transforms**, **Tailwind CSS**, iyo **Particles.js** si uu u abuuro nidaam holographic ah oo u leexanaya dhinac kasta oo mouse-ka loo dhaqaajiyo.

---

## ✨ Awoodaha Boggan (Features)

* 📐 **True 3D Tilt Effect:** Kaadhku wuxuu si smooth ah u raacaa dhaqdhaqaaqa mouse-ka (`perspective: 1000px`), isagoo u tiilanaya dhanka aad farta ama mouse-ka marsiiso.
* 🌟 **Dynamic Interactive Glow:** Waxaa koodhka ku dhex jira `glow-overlay` oo isticmaalaya CSS Variables (`--x`, `--y`) si uu iftiinku u rabo meesha saxda ah ee mouse-ku taagan yahay.
* 🌌 **Live Particle Background:** Background-ka waxaa ka shaqaynaya laybreeriga caanka ah ee *Particles.js* oo abuuraya dhibco cyan ah oo si tartiib ah u dhex sabaynaya shaashadda.
* ✍️ **Real-Time Name Splitter:** Sanduuqa qoraalka ee sare wuxuu kuu ogolaanayaa inaad magacaaga ku qorto, nidaamkuna wuxuu si otomaatig ah u weyneynayaa ereyga hore halka ereyada kalena uu ka dhigayo midabka Cyan-ka ah.
* 📸 **Instant Image Preview:** Waxaad toos u soo upload-gareyn kartaa sawirkaaga adoo isticmaalaya **JavaScript FileReader API**, kaas oo isla markiiba beddelaya avatar-ka caadiga ah.

---

## 🛠️ Tiknoolajiyada la isticmaalay (Tech Stack)

* **HTML5 & Vanilla JavaScript (ES6+)** - Maamulka xisaabta xaglaha (`rotateX`/`rotateY`) iyo xisaabinta xariiqyada kaadhka bartamihiisa.
* **Tailwind CSS** - Maamulka naqshadaynta, blur-ka ka dambeeya kaadhka (`backdrop-blur-xl`), iyo glassmorphism-ka.
* **Particles.js (CDN)** - Loogu talagalay abuurista animations-ka background-ka.
* **Google Fonts** - Font-ka aadka u weyn ee `Syncopate` iyo kan qoraallada caadiga ah ee `Inter`.
